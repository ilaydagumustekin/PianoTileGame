# PianoTileGame

A rhythm-based iOS game inspired by Piano Tiles, developed with SwiftUI. The game challenges players to tap only the black tiles while keeping up with an increasingly fast pace.

## Features

- Built entirely with SwiftUI
- Smooth animations
- Score and best score tracking
- Increasing game speed
- Clean and responsive interface
